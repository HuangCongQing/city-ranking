@ [ChungKing](https://github.com/HuangCongQing/city-ranking)，若fork或star请注明来源


中国城市商业魅力排行榜
>**第一财经**沿用了从2016起设立的**商业资源集聚度、城市枢纽性、城市人活跃度、生活方式多样性和未来可塑性**五大指标.
并维持了原有的算法框架：一级与二级指标的权重以新一线城市研究所专家委员会打分的方式计入，二级指数以下的数据则采用主成分分析法。
